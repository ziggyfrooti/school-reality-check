## Honest Limitations

### Data Lag:
- **Finance data**: FY 2021-22 (18-month lag is normal)
- **Enrollment**: 2023-24 school year (current)
- **Property taxes**: 2024 estimates (verify with county)

### What's Missing:
- Academic performance (test scores, grades)
- Teacher data (student-teacher ratios)
- Program offerings (AP, IB, sports, arts)

### Estimates vs Actual:
- Property taxes are ESTIMATES based on county averages
- Your actual tax depends on:
  - Exact address
  - Township
  - Voter-approved levies
  - Homestead exemptions
- **ALWAYS verify with county auditor before buying**
